One-line install

Get zipgo running.

Pick your platform and paste the command. The script detects your architecture automatically.

macOS
detected
# macOS (Intel & Apple Silicon) curl -fsSL https://install.zipgo.xyz/macos.sh | sh
→ auto-detects arm64 / amd64
Linux
detected
# Linux (amd64 & arm64) curl -fsSL https://install.zipgo.xyz/linux.sh | sh
→ auto-detects arm64 / amd64
Windows
detected
# Windows (PowerShell) iwr https://install.zipgo.xyz/windows.sh -UseBasicP | iex
→ places zipgo.exe in %USERPROFILE%\bin
